# 🔐 TajCars - Admin Panel

This is the **Admin Portal** for TajCars, built with ReactJS. It allows the business owner to manage car listings, view inquiries, and maintain site content without developer support.

## 🧰 Tech Stack

- ReactJS
- Axios
- Tailwind CSS
- Cloudinary (image uploads)
- JWT Auth (optional if added)

## 📦 Admin Features

- Add new car with multiple images
- Edit existing listings
- Delete cars
- View submitted contact/inquiry forms
- Protected dashboard layout

## 🚀 How to Run

```bash
git clone https://github.com/AhmarZamir/Admin-Portal.git
cd Admin-Portal
npm install
npm start
